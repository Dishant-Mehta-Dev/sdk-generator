# Revisions

This page lists the different revisions of the Model Context Protocol. See [Versioning](https://spec.modelcontextprotocol.io/specification/2024-11-05/basic/versioning/) for more information about how the protocol is versioned.

Revisions may be marked as:

-   **Draft**: in-progress specifications, not yet ready for consumption.
-   **Current**: the current protocol version, which is ready for use and may continue to receive [backwards compatible changes](https://spec.modelcontextprotocol.io/specification/2024-11-05/basic/versioning/).
-   **Final**: past, complete specifications that will not be changed.
